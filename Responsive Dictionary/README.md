# EDMTDev (c) 2019

[![EDMTDev](https://preview.ibb.co/nsr2RV/EDMTDev-Logo.png)](https://youtube.com/eddydn71)

Project converter The Online Text Plain English Dictionary (OPTED) to SQLite database and JSon
# New Features!
  - Full A-Z letter convert
  - SQLite database (2 table : Letter and Word)
  - JSON file (array format)
